# ZERO REAL ESTATE HOLDINGS (ZREH)

## Overview
ZERO REAL ESTATE HOLDINGS (ZREH) is a fractional real estate investment platform designed for Kenya.  
It democratizes property investment by allowing users to purchase fractional shares, track portfolios in real time, and engage with gamified features such as XP, challenges, and rewards.

## Objectives
- Democratize real estate investment in Kenya.
- Provide transparency and trust through digital tracking.
- Attract diaspora investors with a modern, engaging platform.
- Build a scalable and secure web application.

## Features
- **User Management**: Sign up, log in, KYC verification, wallet integration (M-Pesa API).  
- **Investment Engine**: Browse properties, purchase fractional shares, track investments.  
- **Gamification**: XP points, challenges, rewards to encourage engagement.  
- **Analytics**: Real-time portfolio tracking with charts and insights.  

## Documentation
Full system documentation is available in the `/docs` folder.  
- [System Documentation (PDF)](./docs/SystemDocumentation.pdf)

## Technology Stack
- **Backend**: Python (Django)  
- **Frontend**: JavaScript, HTML, CSS  
- **Database**: SQLite (development), scalable options for production  
- **Payments**: M-Pesa, Stripe, Flutterwave APIs  

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/calebrogony/ZeroRealEstate.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run migrations:
   ```bash
   python manage.py migrate
   ```
4. Start the server:
   ```bash
   python manage.py runserver
   ```
